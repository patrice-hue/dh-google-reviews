Display Google Business Profile reviews on your WordPress site with shortcodes, Gutenberg blocks, and automatic schema markup output.

== Description ==

A lightweight WordPress plugin that pulls live reviews from Google Business Profile and displays them with configurable layouts (grid, slider, list). Outputs valid AggregateRating and Review schema markup automatically. Supports manual review entry and CSV import for sites without API access.

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/dh-google-reviews/`
2. Activate the plugin through the Plugins screen in WordPress
3. Navigate to Reviews > Settings to configure API connection and display options
